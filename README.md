# TFD+ UC15

## Sistema Desktop para Gerenciamento de Pacientes TFD

Projeto acadêmico desenvolvido para a UC15 do curso Técnico em Desenvolvimento de Sistemas.

## Objetivo do Projeto

O sistema TFD+ foi desenvolvido com foco na organização das informações de pacientes do programa Tratamento Fora de Domicílio (TFD), utilizando Java Swing, MySQL e integração com banco de dados.
A proposta considera situações que podem dificultar o acesso dos pacientes aos atendimentos, especialmente em casos envolvendo idosos, pessoas com dificuldades de comunicação e usuários em situação de maior vulnerabilidade social.


## Tecnologias Utilizadas

- Java
- Java Swing
- MySQL
- JDBC
- Maven
- NetBeans
- IntelliJ IDEA


## Funcionalidades

- Cadastro de pacientes
- Listagem de pacientes
- Integração com banco de dados
- Interface desktop
- Organização de informações do TFD
